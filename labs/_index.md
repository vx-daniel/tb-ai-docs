# Labs: Hands-on Exercises

Use these guided labs to practice ThingsBoard Rule Engine internals. Each lab includes goals, prerequisites, step-by-step actions, success criteria, and example outputs you can compare against.

## Prerequisites

- Running ThingsBoard instance with access to logs
- Ability to modify or import a rule chain
- Access to device credentials to send test telemetry/attributes
- Permission to view server logs (or aggregated logs in your environment)
