# ThingsBoard Rule Engine API: RuleEngineAiChatModelService

## Purpose
Provides AI chat model integration for rule engine nodes, enabling advanced conversational and AI-driven features.

## Key Responsibilities
- Interfaces with AI chat models for message processing and enrichment.
- Used by nodes that require AI-based decision making or text generation.

## Usage Pattern
- Called by AI/ML nodes for chat or NLP tasks.

## Best Practices
- Validate and sanitize AI model inputs and outputs.

## Pitfalls
- Unreliable AI responses can affect rule outcomes.
